# NXT Energy Trading Platform

The world's best energy trading platform operating at a national level.

## Features

1. AI-powered simulation for buyers, offers, and acceptances
2. Digital smart contract management
3. Carbon credit marketplace with fund manager integration
4. Independent Power Producer (IPP) setup and management
5. Financial close and off-take commitment processing
6. Commercial Operation Date (COD) tracking
7. Energy mix management for diverse energy portfolios

## System Components

### Core Services
- Trading Engine
- Market Simulation AI
- Contract Management System
- Carbon Credit Exchange
- IPP Onboarding Suite
- Energy Portfolio Manager

### Technical Stack
- Node.js backend with Express framework
- MongoDB for data persistence
- Redis for caching and queuing
- Ethereum blockchain for smart contracts
- React frontend with Material UI
- WebSockets for real-time trading
- Docker for containerization
- Kubernetes for orchestration
