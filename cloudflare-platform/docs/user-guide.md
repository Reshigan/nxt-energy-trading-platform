# NXT Energy Trading Platform — User Guide

## Getting Started

### Login
1. Navigate to https://et.vantax.co.za
2. Click "Login" on the landing page
3. Enter your email and password
4. You'll be redirected to your role-specific dashboard

### Roles
| Role | Access |
|------|--------|
| Admin | Full platform access, user management, audit logs |
| IPP/Generator | Projects, generation metering, contracts, carbon credits |
| Trader | Order placement, positions, P&L, market data |
| Carbon Fund | Carbon credits, options, fund NAV, registry sync |
| Offtaker | Contracts, invoices, settlement, consumption data |
| Lender | Project finance, disbursements, conditions precedent |
| Grid | Grid connection, system operations, dispatch |

## Dashboard
Your dashboard displays role-specific KPIs:
- **Portfolio Value** — Total value of your holdings
- **Active Contracts** — Number of contracts in execution
- **Open Orders** — Pending buy/sell orders
- **Market Ticker** — Live energy market prices

## Trading
1. Navigate to **Trading** from the top nav
2. Select a market (Solar, Wind, Gas, etc.)
3. Click **New Order** to place a buy or sell order
4. Choose order type: Limit, Market, Stop Loss, or Iceberg
5. Set volume and price (in ZAR cents/MWh)
6. Monitor your open orders and positions

## Contracts
1. Navigate to **Contracts**
2. View existing contracts or click **New Document**
3. Choose document type (PPA, Term Sheet, LOI, etc.)
4. Add signatories and go through the signing flow
5. Track phase progression from Negotiation → Execution → Completion

## Carbon Credits
1. Navigate to **Carbon**
2. View your carbon credit inventory
3. Use tabs: Inventory, Options, Fund, Registry, Tokens, RECs
4. Retire credits, write options, or transfer to other participants

## Settings
- **Profile** — Update contact details
- **Notifications** — Configure email/push preferences
- **Theme** — Switch between dark and light mode
- **Security** — Change password, manage sessions
